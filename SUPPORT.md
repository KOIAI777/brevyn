# Brevyn Community Support

Use the public repository for Community product and development questions. Before opening an issue, search existing issues and remove private course materials, credentials, API keys, and personal data from screenshots and logs.

## Where to Report

- Use the **Bug report** form for reproducible defects.
- Use the **Feature request** form for proposed capabilities or workflow changes.
- Use the **Office/PDF compatibility report** form for preview, parsing, indexing, or source-navigation problems involving a document format.
- Use GitHub private vulnerability reporting for suspected vulnerabilities, exposed credentials, unsafe file handling, or sandbox escapes. Do not post these publicly.

## Community and Official

This repository supports Brevyn Community. Brevyn Official account, subscription, wallet, redemption, and hosted-service questions are outside the scope of public Community issues. Do not post account details or service credentials publicly.

Community maintainers may close reports that cannot be reproduced, contain no actionable detail, or concern code and services that are not part of this repository.
